# Karadul Roadmap

This document outlines the planned features and milestones for Karadul.

## Current Status

**Latest Release:** [v0.1.0-beta.1](https://github.com/karadul/karadul/releases/tag/v0.1.0-beta.1)

## Milestones

### v0.1.0-beta.x (In Progress)

**Goal:** Stabilize Windows support and gather community feedback

- [x] Windows TUN support (Wintun)
- [x] Cross-platform firewall management
- [x] GitHub Actions CI/CD
- [x] Docker support
- [ ] Windows testing and bug fixes
- [ ] Beta user feedback collection
- [ ] Documentation improvements

### v0.1.0 (Stable)

**Goal:** First stable release

- [ ] Windows support stabilized
- [ ] All known beta bugs fixed
- [ ] Complete documentation
- [ ] Homebrew tap published
- [ ] Installation scripts for all platforms

### v0.2.0

**Goal:** Enhanced security and management

- [ ] WireGuard key rotation
- [ ] Certificate-based authentication
- [ ] Web UI for server management
- [ ] Metrics and monitoring dashboard
- [ ] Audit logging

### v0.3.0

**Goal:** Scale and performance

- [ ] Multi-region coordination servers
- [ ] Server clustering
- [ ] Performance optimizations
- [ ] Connection pooling
- [ ] Bandwidth limiting/QoS

### v0.4.0

**Goal:** Advanced networking

- [ ] IPv6 support
- [ ] Subnet routing improvements
- [ ] BGP integration
- [ ] Custom DNS zones
- [ ] Split tunneling

### v1.0.0

**Goal:** Production-ready enterprise features

- [ ] High availability coordination
- [ ] Backup and restore
- [ ] Multi-tenancy
- [ ] Enterprise SSO integration
- [ ] Compliance features
